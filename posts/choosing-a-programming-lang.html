Choosing a Programming Language

So I'm choosing a programming language

What did I value?

The following list is loosely ordered, so the highest item is the most important but not necessarily by that much.

  1. Functional, but doesn't need to be purely functional
    I can't code without higher-order functions or first-class functions.
  2. High-Quality libraries
    I don't want to have to reinvent the wheel, I want to be able to choose to do that.
  3. Good REPL
    After dabbling with Common Lisp, I can't stand working with a slow development loop.
  4. Can produce static binaries
    I find them really useful for smaller programs and scripts.
  5. Lispy
    I love the simplicty of Lisp's syntax.
