Probablistic Data Structures

Main Types

Bloom Filter | Tests if input is definetely not or maybe in a set

HyperLogLog | Counts instances of elements in a multiset

Skip List | Ordered sequence with efficient insertion

Treap | A binary tree where node priorities are randomized

This structure avoids the simple explanations of its value, in contrast to the previous structures. Treaps have the advantage over traditional binary search trees of efficient inseration, deletion and search. Since treap's shape is a product of a random distribution no matter what change occurs, it always has approximately the same efficiency.
